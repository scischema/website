Polymerase Chain Reaction (PCR)

A SciSchema.org Process Schema

Domain: Biology & Biotechnology
Schema: PCR Master Schema

Schemas › Biology & Biotechnology › PCR

The Polymerase Chain Reaction (PCR) is a laboratory technique used to make millions of copies of a specific DNA segment. Starting from a very small amount of DNA, PCR repeatedly heats and cools the sample so that the target DNA is copied over and over again. This makes it possible to detect, analyze, or sequence DNA that would otherwise be too scarce to study.

PCR is commonly used in medical diagnostics, forensic analysis, genetic research, and environmental monitoring.

Properties from PCR

The table below lists the top-level properties of the PCR master schema. Each property may contain more detailed nested fields. For example, reaction_components includes template DNA, primers, DNA polymerase, dNTPs, buffer system, probes, controls, and carryover-prevention details. thermal_cycling_conditions includes denaturation, annealing, extension, cycle number, ramp rate, melting curve analysis, and ddPCR-specific settings.

Property Expected Type Description Required
experiment_metadata Object Administrative and contextual metadata for the experiment. Yes
sample_preparation Object Sample origin, type, extraction, and quality. No
reaction_components Object All reagents used in the PCR reaction. Yes
thermal_cycling_conditions Object Full thermal cycling protocol including initial denaturation, amplification cycles, final extension, and optional hold. Yes
amplicon Object Properties of the amplified DNA fragment. Yes
detection_method Object Method for visualizing or quantifying PCR products. Yes
reaction_volume Number Total PCR reaction volume in µL. Yes
controls Object Experimental controls. No
